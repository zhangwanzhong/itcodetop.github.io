Docker Hub是Docker官方维护的Docker Registry，上面存放着很多优秀的镜像。不仅如此，Docker Hub还提供认证、工作组结构、工作流工具、构建触发器等工具来简化我们的工作。

前文已经讲过，我们可使用docker search 命令搜索存放在Docker Hub中的镜像。本节我们来详细探讨Docker Hub的使用。
