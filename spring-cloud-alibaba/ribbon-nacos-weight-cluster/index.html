在Nacos上，支持集群配置。集群是对指定微服务的一种虚拟分类。集群还是比较有用的，例如：

  • 为了容灾，把指定微服务同时部署在两个机房（例如同城多活【其中1个机房崩溃另一个机房还能顶】、异地多活【防止自然灾害，例如地震什么的】），比如南京机房和北京机房。
    • 调用时，可优先调用同机房的实例，如果同机房没有实例，再跨机房调用。

当然cluster还有很多其他作用，请各位看客自行脑补，本文将围绕上面描述的场景展开。
