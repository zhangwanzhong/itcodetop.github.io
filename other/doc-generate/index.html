总体步骤

  • 整合Swagger，生成Swagger描述端点 /v2/api-docs
  • 使用 swagger2markup-maven-plugin ，将 /v2/api-docs 生成ASCIIDOC文件；
  • 使用 asciidoctor-maven-plugin ，将ASCIIDOC文件转换成HTML；
