Hystrix的超时

如上，Hystrix的默认超时时间是1秒。默认开启超时机制。如需关闭Hystrix的超时，可将xxx.enabled设置为false。

Tips

如有组件跟Hystrix配合使用，一般来讲，建议Hystrix的超时 > 其他组件的超时，否则将可能导致重试特性失效。
