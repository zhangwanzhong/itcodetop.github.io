# 可视化

SkyWalking 原生 UI 提供了一种默认的可视化方案. 它提供了关于总览, 服务, 服务实例, 端点, 追踪和报警相关的观测图, 包括了拓扑图, 依赖图, 热点图等等.

对于大多数开源用户, 这个 UI 方案应该足以满足需求. 当然你还可以使用其他开源的可视化方案. 参考 UI 设置.

我们还了解到, 许多用户已经将 SkyWalking 集成到他们自己的产品中了. 如果你也想将 SkyWalking 集成到自己的产品中, 请使用 SkyWalking 查询协议.
