# 安装

安装过程基于你想要使用哪种探针。如果你对此不了解，请首先阅读概念与设计

重要：请确保被监控的服务器上的系统时间和OAP服务器上的系统时间是相同的。

# 下载官方发行版

  • 后端和UI、Java agent都是Apache官方发行，你可以在Apache SkyWalking 下载页找到它们。

# 各语言agent

  • Java agent。介绍了如何将java agent安装到你的服务中，不需要修改任何代码。

下面的agent和SDK都与SkyWalking的数据格式、传输协议兼容，但是是由第三方维护。你可以到它们的项目仓库中找到对应的发行版，以及如何使用它们的说明。

  • SkyAPM .NET Core agent。可以通过 .NET Core agent的项目文档查看到更详细的信息。
  • SkyAPM Node.js agent。可以通过Node.js服务端agent的项目文档查看到更详细的信息。
  • SkyAPM PHP SDK。可以通过PHP agent项目文档查看到更详细的信息。
  • SkyAPM GO2Sky. 参考 GO2Sky 项目文档获得更多信息.

# Service Mesh

  • Istio
    • SkyWalking on Istio。介绍了如何使用Istio Mixer，并通过适配与SkyWalking一起工作。
  • Envoy
    • 使用 ALS(访问日志服务) 来观测服务网格, 无需使用 Mixer. 按照 文档 来开启此功能.

# Proxy

  • Envoy Proxy
    • 从 Engoy 向 SkyWalking 发送指标数据。介绍了如何使用Metrics 服务 将指标项从Envoy发到SkyWalking。
    • 从 Envoy 发送指标到 Skywalking. 介绍了如何使用 Metrics service 向 SkyWalking 发送指标.

# 安装后端

从 后端和 UI 设置文档中可以了解到在不同的场景下如何对后端服务进行配置，以及如何开启高级特性。
